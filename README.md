# FRONTEND-STUDY

### JS 공부 시 npm 실행 방법
1. node.js 설치
2. nvm-windows 구글 검색 후 첫 링크
3. 스크롤을 내려 Download now 클릭
4. nvm-setup.zip 다운
5. nvm 설치
6. npm init -y
7. npm install parcel-bundler -D
8. "scrpit" 내 "test" 부분을 지우고 "dev": "parcel index.html", "build": "parcel build index.html" 작성
9. index.html 생성
10. npm run dev로 실행
